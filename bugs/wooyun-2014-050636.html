code 区域
function image($field, $value) {


    $value = str_replace(array("'",'"','(',')'),'',$value);


    return trim($value);


}



过滤了"'"、"("、")",但是呢 我们知道当开启了GPC的时候，单引号会被转义 '-->\'

明白了吧? image函数过滤了单引号，假设我们提交的数据恰巧经过了image函数,则单引号被过滤了，留下"\"，那么这个"\"将会吃掉一个单引号,造成注入
